# Biomod 网站使用说明

## AI 助手功能

网站集成了智能AI助手功能，**无需任何配置即可直接使用**！

### 特点

- 🚀 **即开即用**：无需API密钥，打开网页即可使用
- 🧠 **智能回复**：针对生物医学项目内容提供专业回答
- 💬 **多种交互方式**：支持气泡按钮、侧边栏和底部输入框
- 🎯 **项目专属**：了解网站各个页面内容，提供导航帮助

### 使用方式

你可以通过以下三种方式与AI助手交互：

1. **气泡按钮**：点击页面右下角的💬气泡按钮打开聊天侧边栏
2. **底部输入框**：在首页底部的输入框中直接提问
3. **展开聊天**：点击"展开聊天"按钮打开完整的聊天界面

### AI助手能力

AI助手可以帮助你：

- 📖 **了解项目**：介绍Biomod项目的背景、目标和特色
- 🔬 **解释设计**：说明技术方案和系统架构
- 🧪 **介绍方法**：解释实验流程和研究方法
- 📊 **展示结果**：引导查看研究成果和数据分析
- 👥 **认识团队**：介绍团队成员和背景信息
- 🗺️ **网站导航**：帮助找到相关页面和内容

### 示例对话

你可以尝试这些问题：

- "这个项目是关于什么的？"
- "团队有哪些成员？"
- "实验方法是什么？"
- "项目取得了什么结果？"
- "如何查看设计方案？"

## 技术说明

- 🔧 **纯前端实现**：无需服务器，完全基于JavaScript
- 💾 **本地运行**：所有处理都在浏览器中完成
- 🔒 **隐私安全**：不会收集或传输用户数据
- 📱 **响应式设计**：支持桌面和移动设备
